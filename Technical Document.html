What is HTML

HTML stands for Hyper Text Markup Language.

HTML is the standard markup language for creating Web pages.

HTML describes the structure of a Web page.

HTML consists of a series of elements.

HTML elements tell the browser how to display the content.

<!DOCTYPE html> <html>...</html>

HTML Syntax

HTML documents consist of nested HTML elements.

An element starts with a start tag and ends with an end tag.

The content is placed between the start and end tags.

<p>This is a paragraph.</p> <h1>This is a heading</h1>

HTML Tags

HTML tags are used to define elements on the web page.

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (link)
  • <div>: Container
  • <img>: Image

HTML Lists

Lists are used to group related items together.

There are two types of lists: ordered and unordered.

  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item
  • You can nest lists inside each other
  • Lists help organize content meaningfully

What is CSS

CSS stands for Cascading Style Sheets.

CSS describes how HTML elements are to be displayed on screen.

CSS saves a lot of work. It can control the layout of multiple web pages at once.

body { font-family: Arial; } h1 { color: blue; }
